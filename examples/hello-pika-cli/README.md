# hello-pika-cli

1. Install dependencies: `npm install`
2. Package executables: `npm run build`
3. Run the executable corresponding to your architecture:

```shell
# MacOS
$ ./pkg/bin/hello-pika-macos
Hello Pika ʢ◉ᴥ◉ʡ

$ ./pkg/bin/hello-pika-macos Roger
Hello Roger ʢ◉ᴥ◉ʡ
```
